# Git 커밋 메시지

## 커밋 1: 타임라인 개선 및 통계 기능 향상

### 제목
```
feat: 시간 흐름 타임라인 개선 및 통계 기능 리팩토링
```

### 내용
```
- 시간 흐름 타임라인을 GitHub 스타일 정사각형 그리드로 개편
  - 1초 단위 정밀 시간 표시
  - 블록 채도 시스템 구현 (1시간, 30분, 15분, 1분, 30초, 15초, 1초)
  - 활동 선택 및 하이라이트 기능 추가
  - 시간 범위 표시 (시작 시간 ~ 종료 시간, 전체 시간)
  - 활동 범례 하단 배치
  - 블록 채도 범례 상단 배치

- 활동별 시간 분포 차트 개선
  - 활동 리스트 추가 (각 활동별 시간, 기록 수, 비율 표시)
  - 총 활동 수 및 전체 시간 헤더에 표시
  - 바 차트 레이블 개선 (막대 위 시간 표시)
  - 툴팁에서 초 단위 제거

- 통계 계산 로직 리팩토링
  - utils/statistics.ts 유틸리티 파일 생성
  - calculateTimeStats, calculateActivityStats 함수 분리
  - MISC_ACTIVITY, MIN_DURATION_MS 상수 중앙화
  - 컴포넌트 간 코드 중복 제거

- 기록 섹션 개선
  - 시간 통계 표시 (전체 시간, 집중 시간, 잡동사니 시간)
  - 반응형 레이아웃 개선 (작은 화면에서 깔끔한 줄바꿈)
  - 복사 버튼 상단 이동
  - 통계 디자인을 타임라인과 통일

- UI/UX 개선
  - 삭제 버튼 호버 시 회전 애니메이션 제거
  - 시간 표시 형식 통일 (초 단위 정확한 표시)
```
